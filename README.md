# Blockchain Technology in Capital Markets - Thesis Code Repository

This repository contains the Python source code and raw data files supporting the bachelor's thesis:
**Blockchain Technology in Capital Markets: Maturity Assessment, Trend Cluster Analysis, and Scenario Planning to 2030**
Szymon Hajbowicz, Kozminski University, Warsaw, 2026

## Contents
- "charts.py" — Figures 1–4 (market cap, DLT issuance, CASP growth, VC funding)
- "survey_charts.py" - Figures 5–8 (perception survey visualisations)
- "trendmap_final.py" — Figure A1 (thematic coding structure diagram)
- "CoinGecko-GlobalCryptoMktCap-2026-05-07.csv" — raw market cap data (CoinGecko, 2026)
- "perception survey-c8786e5b.csv" — raw survey data (n=38, May 2026)
- "Appendix_CodingTable.csv" — thematic coding table (97 codes, 6 categories, 3 clusters)

## Note
Market capitalisation statistics (trough, peak, Q1 2026 values) were computed directly from the CoinGecko CSV using standard MIN/MAX functions in Excel and are not included as a separate script.
